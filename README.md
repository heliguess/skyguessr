<h1 align="center">
  SkyGuessr
</h1>

SkyGuessr is a GeoGuessr-like game for Hypixel SkyBlock!
You will be presented with a panoramic image of somewhere in SkyBlock, and the goal is to figure out where you are on the map, down to the block.
For every guess, you will get up to 5000 points based on how close you are, each round having 5 maps.

The image will be greyscaled and with a render distance of 4 chunks in order to make it an actual challenge
